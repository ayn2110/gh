### 藥草
船子草繃帶 ( 所有繃帶的基底，以下簡稱繃帶 ) : 可用於包紮簡單的擦傷、扭傷。
菸草繃帶 ( 菸草葉+繃帶 ) / 百合繃帶 ( 未知藥草/繃帶 ) : 解毒用，適合用於蛇/蟲/魟魚毒咬傷。
灰燼繃帶 ( 灰燼+繃帶 ) : 用於撕裂傷，如野人/美洲豹/食蟻獸/鱷魚之攻擊。
蜂蜜繃帶 ( 蜂蜜+繃帶 ) : 好像也適用於撕裂傷或傷口感染。
傷口感染 : 先用蛆，再用上述任意繃帶 ( 應該用一般繃帶即可? )。
巨型敷料：繃帶 + 巨型食鳥鳥灰（可治癒感染傷口）

### 燉煮/烘烤/風乾/煙燻
燉煮 vs 烘烤：烘烤食物需要時間（尤其是優質肉類），燉煮則非常快捷，缺點是燉菜不方便攜帶。
風乾 vs 煙燻：煙燻比風乾快，但需要點燃營火，而且煙燻爐還能防止營火被雨水澆熄。風乾/煙燻食物的保存時間是一般食物的三倍。有些植物風乾後也能帶來意想不到的驚喜。


### 鍋類
#### 終極骨頭湯
生一小堆篝火
將椰子半塊（又稱椰子碗）拖放到其兩個插槽之一上
確保它裝滿了水。
將一塊骨頭拖進碗裡
點燃篝火
等它煮沸後就可以食用了。

#### 開水
1個裝滿水的椰子水壺
在營火兩側各放兩個椰子碗，以達到最佳效果。
將水倒入每個碗中
煮沸幾分鐘
如果你想收集水，請使用第二個椰子水壺。

#### 抗蛇毒血清輸注
一碗水
紫花/心形葉草本植物或菸草葉
煮沸幾分鐘

#### 發熱輸液
一碗水
苦參花（開紅花的大灌木）
煮沸幾分鐘
如果發燒持續不退，一定要休息。

#### 抗寄生蟲輸液
一碗水
1個杯狀菇類或刺果番荔枝
煮沸幾分鐘
生吃蘑菇比生吃蘑菇能殺死更多寄生蟲。

#### 肉燉菜
1塊任何新鮮肉類
一碗水
像往常一樣煮肉。
更有營養，而且烹飪速度很快，用它來掩蓋吃昆蟲帶來的精神萎靡感。
